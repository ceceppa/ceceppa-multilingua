Function cml_get_the_link

return link of current page in according to selected $language, so if $result != current language this function will return its translation link.

Located at utils.php
Parameters summary
stdObject $result - language object ( i.e. CMLLanguage::get_default() ) used to translate current link
boolean $linked = true - true, return linked translation, false return homepage link
boolean $only_existings = false - return linked post only if it exists, otherwise return blank link
boolean $queried = false - use get_queried_object_id instead of get_the_ID return string
